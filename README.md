# Osushenie Control

Osushenie Control is an internal web system for managing construction projects, tasks, employees, and work progress.

The system is designed for a construction company workflow where administrators, chief engineers, and foremen need a single place to control objects, assign tasks, track progress, and keep work history organized.

## Purpose

The project aims to replace scattered communication and manual tracking with a structured backend system for construction object management.

Main focus areas:

- construction objects
- task assignment
- employee roles
- work progress tracking
- standardized task templates
- activity history
- notifications
- future AI-based progress monitoring

## User Roles

- **Administrator** - manages users, objects, and system data
- **Chief Engineer** - controls work progress and supervises objects
- **Foreman** - works with assigned objects and tasks
